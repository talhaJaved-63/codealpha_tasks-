 CodeAlpha Artificial Intelligence Internship Projects

The projects demonstrate practical applications of AI, Natural Language Processing (NLP), Computer Vision, and API integration using Python and related technologies.

📌 Project 1: Language Translation Tool

Overview

A web application that translates text from one language to another using a translation API. Users can enter text, select source and target languages, and receive an instant translation.

 Features
* Translate text between multiple languages
* User-friendly interface
* Source and target language selection
* Fast API-based translation
* Copy translated text (optional)
* Text-to-Speech support (optional)

 Technologies Used
* Python
* Flask
* HTML
* CSS
* JavaScript
* Translation API

 🤖 Project 2: FAQ Chatbot

 Overview
An NLP-powered chatbot that answers user questions by matching them with a predefined FAQ dataset using text preprocessing and similarity matching techniques.

Features
* Interactive chatbot interface
* FAQ-based responses
* NLP preprocessing
* Cosine Similarity matching
* Fast and accurate answer retrieval

 Technologies Used

* Python
* Flask
* NLTK / SpaCy
* Scikit-learn
* HTML
* CSS
* JavaScript


 🎯 Project 3: Object Detection and Tracking

 Overview

A real-time computer vision application that detects and tracks objects using a webcam or video file with a pre-trained YOLO model and tracking algorithms.

 Features
* Real-time object detection
* Webcam and video support
* Bounding boxes with labels
* Object tracking using SORT/Deep SORT
* Live tracking IDs

 Technologies Used
* Python
* OpenCV
* YOLO
* SORT / Deep SORT
* NumPy

 🛠️ Installation

1. Clone this repository.

bash
git clone https://github.com/your-username/CodeAlpha-AI-Projects.git

2. Navigate to the repository.

bash
cd CodeAlpha-AI-Projects

3. Install the required dependencies for the project you want to run.

bash
pip install -r requirements.txt

4. Run the desired project from its respective folder.

Example:
bash
cd Language-Translation-Tool
python app.py


or

```bash
cd FAQ-Chatbot
python app.py
```

or

```bash
cd Object-Detection-and-Tracking
python main.py
```
